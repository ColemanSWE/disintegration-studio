# Disintegration Studio

Photo effects and video glitch art studio.

## Features

- **Photo Effects**: Apply artistic effects to photos and camera feed
- **Datamosh Studio**: Create glitch art by corrupting video codecs

## Development

```bash
# Install dependencies
pnpm install

# Run dev server
pnpm dev

# Build
pnpm build
```

## Tech Stack

- React 19
- TypeScript
- Vite
- FFmpeg (for datamoshing)
- Canvas API (for effects)
